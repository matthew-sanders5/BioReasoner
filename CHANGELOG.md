# Changelog

## v0.1.0

Initial public release.

Included:
- Deterministic symbolic reasoning engine (forward-chaining rules, contradictions, provenance)
- YAML scenario format and scenario loader
- LLM evaluation harness (prompting, parsing, scoring vs engine ground truth)
- CLI: engine, eval-llm, batch-eval, analyze, aggregate-replicates
- Providers: stub (offline), ollama (optional), openai (optional)
- Basic tests and CI workflow
